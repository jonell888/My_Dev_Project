โดย LINE OA สามารถทำงานได้ ดังนี้
1. มี Richmenu ที่สร้างด้วย Messanging API มีการใช้ Richmenu Swich Action สามารถเลือกสลับเมนูไปมาได้
2. มี Chatbot ซึ่งกำหนดเงื่อนไขดังนี้ 
2.1 "LINE Dev Camp" ให้ตอบกลับมาว่า "I am a LINE developer." พร้อมทั้งส่ง Imagemap ตามที่กำหนดกลับมาด้วย ซึ่งการตอบนี้เกิดจากเงื่อนไขที่กำหนดใน Webhook 
2.2 หากพิมพ์คำทักทาย เช่น สวีดัส, สวัสดี, Hello ให้ตอบกลับเป็นการทักทาย เช่น สวัสดีครับ หรือ สวัสดีค่ะ โดยระหว่างการรอคำตอบให้แสดง Loading Animation ด้วย ซึ่งการตอบนี้เกิดจาก Intent ที่กำหนดใน Dialogflow
3. ใน Richmenu มี 1 ปุ่มที่สามารถเปิด LIFF ขึ้นมา แล้วมีผลลัพธ์ดังนี้
3.1 ในหน้าจอให้แสดง รูป Profile และชื่อของผู้ใช้งานที่เปิด LIFF ได้ 
3.2 ในหน้าจอให้มีปุ่มสำหรับส่งข้อความกลับไปในห้อง Chat ในรูปแบบ  Flex Message ตามที่กำหนด
3.3 ในหน้าจอให้มีปุ่มสำหรับ Share Flex Message ที่ใช้ในข้อ 3.2 ให้กับผู้ใช้คนอื่นๆ ได้
